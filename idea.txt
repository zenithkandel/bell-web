AUTOMATIC SCHOOL / COLLEGE BELL SYSTEM
MASTER DESIGN & PROPOSAL DOCUMENT

1. PROJECT OVERVIEW
--------------------------------------------------------
This project is a fully automated, production-level school/college bell system designed to eliminate manual bell operation and provide precise, reliable, and intelligent bell scheduling with audio announcements.

The system operates completely offline, ensuring high reliability even in environments without internet connectivity. It integrates accurate timekeeping, flexible scheduling, emergency alert capabilities, and a user-friendly configuration interface.

--------------------------------------------------------

2. OBJECTIVES
--------------------------------------------------------
- Automate bell ringing without human intervention
- Ensure high accuracy timing using RTC
- Provide flexible scheduling (weekly, seasonal, special days)
- Deliver clear audio announcements through existing PA systems
- Enable offline configuration using a mobile interface
- Ensure long-term reliability and minimal maintenance

--------------------------------------------------------

3. CORE FEATURES
--------------------------------------------------------

SYSTEM FEATURES:
- Fully Offline Operation (No Internet Required)
- 7-Day Programmable Schedule
- Winter / Summer Preset Timetable Switching
- Special Day / Event Schedule Presets
- Priority-Based Schedule Resolution:
  Emergency > Special Day > Seasonal > Weekly

TIME MANAGEMENT:
- High Accuracy RTC (DS3231)
- Automatic Time Sync via Mobile (Offline Hotspot)
- RTC Battery Backup (Up to 5 Years)
- RTC Invalid Time Detection

AUDIO SYSTEM:
- MicroSD Card Based Audio Storage
- MP3 / WAV Audio File Support
- Preloaded Bell & Announcement Sounds
- External PA Amplifier Compatibility
- Clean Line-Level Output via DAC
- Audio Fallback System (Default Beep)

USER INTERFACE:
- 16x2 LCD Display
- Time & Next Bell Display
- System Status Display (RUN / CONFIG / EMERGENCY)
- LED Indicators (Power / Status / Emergency)

CONTROL SYSTEM:
- Manual Next Bell Trigger Button
- Emergency Siren Button (Long Press Protection)
- Config Mode Button (Hotspot Activation)
- System Enable / Disable Switch
- Button Safety Logic (Debounce + Hold Detection)

CONFIGURATION SYSTEM:
- Offline WiFi Hotspot Mode
- Mobile-Friendly Web Dashboard
- Schedule Editing & Saving
- Time Synchronization

RELIABILITY FEATURES:
- Auto Resume After Power Failure
- Missed Bell Handling Logic
- SD Card Health Check
- SD Card Removal Detection
- Audio File Missing Handling
- Watchdog Timer for Auto Recovery

UPDATE SYSTEM:

POWER & SAFETY:
- Industrial AC-DC Power Module
- Fully Isolated Power Design
- Fuse Protection
- MOV Surge Protection

SYSTEM ARCHITECTURE FLOW:

AC INPUT (230V)

AC-DC SMPS (HLK-PM5)
   ↓
Power Distribution (5V / 3.3V)

ESP32-S3 MAIN CONTROLLER

            ↓
